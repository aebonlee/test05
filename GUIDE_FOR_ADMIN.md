## 1. GitHub 가입 및 사용법

### GitHub이란?

- 코드를 저장하고 관리하는 **온라인 저장소**입니다
- 여러 사람이 같은 코드를 함께 수정할 수 있습니다
- 우리 사이트의 코드가 여기에 올라가 있고, 코드를 수정하면 사이트에 자동 반영됩니다

### 가입하기

1. https://github.com 접속
2. 우측 상단 **Sign up** 클릭
3. 이메일, 비밀번호, 사용자명(영문) 입력
4. 이메일로 온 인증 코드 입력하면 가입 완료

### 저장소 권한 받기

가입 후 사이트 코드를 수정하려면 **권한**이 필요합니다.

1. 가입한 **GitHub 사용자명**을 관리자(wksun999)에게 알려주세요
2. 관리자가 협력자(Collaborator)로 초대합니다
3. GitHub에서 초대 이메일이 오면 **Accept invitation** 클릭
4. 이제 코드를 수정하고 반영할 수 있습니다

### GitHub 웹사이트에서 할 수 있는 것

- https://github.com/SUNWOONGKYU/AI_Study_Circle 에서 코드 확인
- 누가 언제 무엇을 수정했는지 기록(커밋 히스토리) 확인
- 간단한 텍스트 수정은 웹에서도 가능 (파일 클릭 → 연필 아이콘)

## 2. Git 설치 및 사용법

### Git이란?

- 코드의 변경 이력을 관리하는 **도구**입니다
- GitHub에 코드를 올리고(push) 내려받는(pull) 데 사용합니다

### 설치하기

1. https://git-scm.com/download/win 접속
2. **Download for Windows** 클릭하여 설치 파일 다운로드
3. 설치 진행 — 모든 옵션 기본값 그대로 **Next** 클릭
4. 설치 확인: 아무 폴더에서 **마우스 오른쪽 클릭** → **"Git Bash Here"** 메뉴가 보이면 성공

### 최초 설정 (1회만)

Git Bash를 열고 아래 두 줄 입력 (본인 정보로 수정):

```bash
git config --global user.name "본인이름"
git config --global user.email "본인이메일@gmail.com"
```

### 알아야 할 명령어 (이것만 알면 됨)

| 명령어 | 의미 | 언제 쓰나 |
|--------|------|-----------|
| `git clone 주소` | 코드 내려받기 | 최초 1회 |
| `git pull` | 최신 코드로 업데이트 | 수정 전 항상 |
| `git add .` | 수정한 파일 선택 | 수정 후 |
| `git commit -m "설명"` | 수정 기록 남기기 | add 후 |
| `git push` | 사이트에 반영 | commit 후 |
| `git status` | 현재 상태 확인 | 아무 때나 |

## 3. VS Code 설치 및 사용법

### VS Code란?

- 코드를 편하게 수정할 수 있는 **무료 편집기**입니다
- 전세계 개발자들이 가장 많이 사용하는 도구입니다
- 코드 수정 + Git 명령어 실행을 **한 프로그램 안에서** 다 할 수 있습니다

### 설치하기

1. https://code.visualstudio.com 접속
2. **"Download for Windows"** 클릭하여 설치
3. 설치 중 아래 옵션 **모두 체크**:
   - "Add to PATH"
   - "Open with Code" 관련 항목 2개
4. 설치 완료

### 한국어로 바꾸기

1. VS Code 실행
2. **Ctrl + Shift + X** 누르기 (확장 마켓플레이스 열기)
3. 검색창에 `Korean` 입력
4. **Korean Language Pack** 설치 → 재시작하면 한국어로 바뀜

### 프로젝트 열기

1. VS Code 실행
2. **파일** → **폴더 열기** → 클론한 `AI_Study_Circle` 폴더 선택
3. 왼쪽에 파일 목록이 나타남 → 파일 클릭하면 열립니다

### 자주 쓰는 단축키

| 단축키 | 기능 |
|--------|------|
| **Ctrl + S** | 저장 (수정 후 반드시!) |
| **Ctrl + F** | 현재 파일에서 텍스트 찾기 |
| **Ctrl + H** | 찾아서 바꾸기 |
| **Ctrl + Shift + F** | 모든 파일에서 찾기 |
| **Ctrl + Z** | 되돌리기 (실수했을 때) |

### VS Code 안에서 Git 명령어 사용하기

Git Bash를 따로 열 필요 없이 VS Code 안에서 바로 실행할 수 있습니다.

1. 상단 메뉴 **터미널** → **새 터미널** (또는 **Ctrl + `**)
2. 화면 아래에 터미널 창이 열림
3. 여기에 `git pull`, `git add .`, `git commit`, `git push` 등 입력

### 명령어 없이 클릭으로 Git 사용하기

1. 왼쪽 사이드바에서 **세 번째 아이콘** (가지 모양 = 소스 제어) 클릭
2. 수정된 파일 목록이 표시됨
3. 파일 옆 **+** 버튼 클릭 → 스테이징 (`git add`와 같음)
4. 상단 입력창에 수정 내용 적기 (예: "문구 수정")
5. **체크(V) 버튼** 클릭 → 커밋
6. 하단 상태바에서 **동기화** 클릭 → 푸시

이 방법이면 **명령어 없이 클릭만으로** 수정 → 반영 가능합니다!

### 추천 확장 프로그램

**Ctrl + Shift + X**로 확장 마켓플레이스에서 검색하여 설치:

| 이름 | 설명 |
|------|------|
| **Korean Language Pack** | 한국어 지원 |
| **Live Server** | 수정 결과를 push 전에 내 PC에서 미리 확인 |

**Live Server 사용법:** 설치 후 `index.html` 열기 → 우측 하단 **"Go Live"** 클릭 → 브라우저에서 미리보기 → 코드 수정하고 Ctrl+S 하면 자동 새로고침됨

## 4. 사이트 코드 내려받기 (클론)

이 단계는 **최초 1회만** 하면 됩니다.

### 방법 A: Git Bash 사용

1. 원하는 위치에 폴더를 만듭니다 (예: `C:\Projects`)
2. 해당 폴더에서 **마우스 오른쪽 클릭** → **"Git Bash Here"**
3. 아래 명령어 입력:

```bash
git clone https://github.com/SUNWOONGKYU/AI_Study_Circle.git
```

### 방법 B: VS Code 사용

1. VS Code 실행 → **Ctrl + Shift + P** → `Git: Clone` 입력 → 선택
2. 아래 주소 붙여넣기:

```
https://github.com/SUNWOONGKYU/AI_Study_Circle.git
```

3. 저장할 폴더 선택 → 자동으로 클론 + 폴더 열기

## 5. 코드 수정하고 사이트에 반영하기

### 파일 구조

```
AI_Study_Circle/
├── index.html          ← 메인 페이지 (가장 자주 수정)
├── admin.html          ← 관리자 페이지
├── css/
│   └── style.css       ← 디자인/스타일
├── js/
│   ├── main.js         ← 메인 페이지 동작
│   ├── admin.js        ← 관리자 페이지 동작
│   ├── supabase-config.js  ← DB 연결 설정
│   └── animations.js   ← 애니메이션 효과
└── supabase/
    └── schema.sql      ← DB 구조 (참고용)
```

### 작업 순서

```
1. git pull                          ← 최신 코드 받기 (항상 먼저!)
2. VS Code에서 파일 수정 + Ctrl+S 저장
3. git add .                         ← 수정 파일 선택
4. git commit -m "수정 내용 설명"     ← 기록 남기기
5. git push                          ← 사이트에 반영
6. 1~2분 후 사이트에서 Ctrl+Shift+R  ← 확인
```

## 6. 자주 수정하는 항목 가이드

### 텍스트 수정 (index.html)

VS Code에서 `index.html` 열고 **Ctrl + F**로 검색:

| 수정 항목 | 검색어 |
|-----------|--------|
| 메인 타이틀 | `자유롭지만 스마트하게` |
| 서브 타이틀 | `바이브 코딩, AI 활용` |
| 가입 멤버 수 | `50+` |
| 소개 섹션 | `이런 모임입니다` |
| 타임라인 | `멤버 모집 중` |

### 디자인 수정 (css/style.css)

상단에 색상 변수가 정의되어 있습니다:

```css
--accent-cyan: #00d4ff;    /* 메인 강조색 (하늘색) */
--accent-purple: #a855f7;  /* 보조 강조색 (보라) */
--accent-pink: #f43f5e;    /* 경고/핑크 */
```

## 7. admin 페이지 사용법

**코드 수정 없이** admin 페이지에서 할 수 있는 작업들:

| 기능 | 위치 |
|------|------|
| 모임 등록/수정/비활성화 | 모임 관리 탭 |
| 장소 등록/수정/삭제 | 장소 관리 탭 |
| 멤버 목록 조회/검색 | 멤버 관리 탭 |
| 모임 참여자 명단 조회 | 모임 관리 > 보기 버튼 |

**접속 방법:**
1. 사이트에서 관리자 계정으로 로그인
2. 우측 상단 이름 클릭 → **관리자 페이지** 클릭

> 모임/장소/멤버 관리는 코드 수정 없이 admin 페이지에서 가능합니다!

## 8. 문제가 생겼을 때

### push가 거부될 때

다른 사람이 먼저 수정한 경우입니다. `git pull` 후 다시 `git push`

### 충돌(conflict)이 발생했을 때

```bash
git merge --abort
git stash
git pull
```

그 후 다시 수정하면 됩니다.

### 내 수정을 전부 취소하고 싶을 때

```bash
git checkout .
```

커밋 전에만 가능합니다.
